# Tiktak :rocket:

Quick and simple Git Cli Tool. Allows you to quickly and easily manage your git repository.

## Requirements

* [Git](https://git-scm.com/)
* [Node.js](https://nodejs.org/)

## Installation

```
npm i -g tiktak
```

or if you prefer to use npx:

```
npx tiktak
```

## Usage

```
tiktak
```
